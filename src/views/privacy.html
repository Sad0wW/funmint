Политика конфиденциальности

Последнее обновление: 21 августа 2025 года

Настоящая Политика конфиденциальности регулирует порядок сбора, хранения и использования персональных данных пользователей сервиса FunMint (далее — «Сервис»), предоставляющего платный функционал автоматизации действий на FunPay.

1. Какие данные мы собираем

  • Email: используется для авторизации, восстановления доступа и отправки уведомлений.
  • Cookies и IP-адрес: применяются для обеспечения корректной работы Сервиса, аналитики и безопасности.
  • golden_key (cookie FunPay): предоставляется пользователем для авторизации и выполнения автоматизации на FunPay. Хранится в зашифрованном виде, используется только внутри FunMint, не передается третьим лицам.
  • Информация о платежах: обрабатывается платёжными системами platega.io и CrystalPay, мы получаем только статусы и факты платежей.

2. Для чего мы используем данные

  • Для авторизации пользователей и обеспечения доступа к Сервису.
  • Для рассылки сервисных писем (например, при входе в аккаунт).
  • Для предоставления автоматизации действий на FunPay через golden_key.
  • Для обработки платежей и управления подписками.
  • Для анализа и улучшения качества работы Сервиса.

3. Хранение и защита данных

Мы применяем организационные и технические меры для защиты данных:

  • хранение golden_key и других чувствительных данных в зашифрованном виде;
  • ограничение доступа сотрудников и подрядчиков по принципу наименьших привилегий;
  • шифрование каналов передачи данных (TLS);
  • мониторинг безопасности и аудит доступа.

Несмотря на предпринимаемые меры, важно учитывать, что компрометация golden_key может дать доступ к аккаунту FunPay, и пользователь обязан обеспечивать его сохранность.

4. Передача данных третьим лицам

Мы не продаём и не передаём персональные данные третьим лицам, за исключением:

  • платёжных провайдеров (platega.io, CrystalPay) для обработки платежей;
  • случаев, прямо предусмотренных законом.

golden_key никогда не передаётся третьим лицам.

5. География обработки данных

Обработка данных может происходить на серверах, расположенных в различных странах. Мы гарантируем соблюдение необходимых мер защиты при трансграничной передаче данных.

6. Права пользователей

Каждый пользователь имеет право:

  • запросить доступ к своим данным;
  • изменить или обновить их;
  • удалить данные (включая golden_key);
  • ограничить обработку данных;
  • отозвать согласие на обработку.

Для реализации прав обращайтесь по адресу: abuse@funmint.ru

7. Изменения политики

Мы оставляем за собой право обновлять настоящую Политику конфиденциальности. Новая редакция публикуется на сайте с указанием даты последнего обновления.

8. Контакты

По всем вопросам, связанным с обработкой данных, обращайтесь: abuse@funmint.ru
